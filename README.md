# Node.js PDF Service

Node.js PDF Service

### Requirements
- [protobuf](https://grpc.io/docs/protoc-installation)

### Generate typescript proto schema
- `make generate`

### Environment
- `PORT=50010`
- `SPACES_BUCKED=`
- `SPACES_CDN_URL=`
- `SPACES_KEY=`
- `SPACES_REGION=`
- `SPACES_SECRET=`
- `SPACES_URL=`
